## 20. IMPORTANT NOTES

### General Warnings

⚠️ **Security Considerations:**
- Many old SVN repositories have been archived
- Backdoors may exist in unofficial sources
- Always verify code before deployment
- Review database scripts carefully
- Test in isolated environment first

⚠️ **Version Compatibility:**
- Lin.bin files are version-specific
- Client and server versions must match
- Database schemas vary by version
- Some features require specific client builds

⚠️ **Legal Considerations:**
- Official NCSoft client required (separate from server)
- Server emulators are for educational purposes
- Respect intellectual property rights
- Private servers operate in legal grey areas

### Resource Quality

✅ **Most Reliable Sources:**
- **Lineage45.com** - Comprehensive Chinese resource hub
- **TodayNC.com** - Working Korean code examples
- **L1Justice** - Modern English development
- **GitHub Official Repos** - Version-controlled source code

⚠️ **Use with Caution:**
- Old tutorial sites with broken links
- Unofficial source downloads
- Unverified database scripts
- Modified clients from unknown sources

### Community Etiquette

- Contribute back to communities that help you
- Share discoveries and fixes
- Document your modifications
- Help newcomers when possible
- Respect community rules and guidelines

### Update Frequency

- **GitHub Repos:** Active development, check regularly
- **Forum Posts:** Varies by community, some very active
- **Tutorial Sites:** Often outdated, verify information
- **IRC/Discord:** Real-time, most current information
